# Candlestick Patterns Reference

Quick visual reference for all candlestick patterns.

## Single Candle Reversal Patterns

### Hammer / Hanging Man

```
    │
    │
   ┌┴┐    Hammer (at support = bullish)
   └─┘    Hanging Man (at resistance = bearish)
    │
    │
    │
```

- Small body at top
- Long lower wick (2x+ body length)
- Little to no upper wick

### Inverted Hammer / Shooting Star

```
    │
    │
    │
   ┌─┐    Inverted Hammer (at support = bullish)
   └┬┘    Shooting Star (at resistance = bearish)
    │
```

- Small body at bottom
- Long upper wick (2x+ body length)
- Little to no lower wick

### Doji

```
    │         │         │
   ─┼─       ─┼─       ─┼─
    │         │         │
              │
 Standard   Long-leg  Dragonfly
```

- Open ≈ Close (tiny/no body)
- Shows indecision
- Reversal signal at extremes

## Multi-Candle Reversal Patterns

### Bullish Engulfing

```
   ┌─┐
   │▓│    Red candle
   └─┘
  ┌───┐
  │   │   Green candle completely engulfs
  │   │
  └───┘
```

### Bearish Engulfing

```
  ┌───┐
  │   │   Green candle
  │   │
  └───┘
   ┌─┐
   │▓│    Red candle completely engulfs
   └─┘
```

### Morning Star (Bullish)

```
  ┌───┐           ┌───┐
  │▓▓▓│   ┌─┐     │   │
  │▓▓▓│   └─┘     │   │
  └───┘           └───┘

  Red    Small   Green
```

### Evening Star (Bearish)

```
  ┌───┐           ┌───┐
  │   │   ┌─┐     │▓▓▓│
  │   │   └─┘     │▓▓▓│
  └───┘           └───┘

 Green   Small    Red
```

## Continuation Patterns

### Three White Soldiers

```
        ┌───┐
      ┌───┐ │
    ┌───┐ │ │
    │   │ │ │
    └───┘ │ │
      └───┘ │
        └───┘
```

Three consecutive bullish candles, each closing higher.

### Three Black Crows

```
    ┌───┐
    │▓▓▓│───┐
    │▓▓▓│▓▓▓│───┐
    └───┘▓▓▓│▓▓▓│
      └───┘▓▓▓│
        └───┘
```

Three consecutive bearish candles, each closing lower.

| -------- | --------------------------------- |
| ⭐⭐⭐   | Engulfing, Morning/Evening Star   |
| ⭐⭐     | Hammer, Shooting Star, 3 Soldiers |
| ⭐       | Doji, Piercing, Dark Cloud        |

## Key Rules

1. **Location matters**: Patterns at key S/R levels are most reliable
2. **Confirmation**: Wait for next candle to confirm
3. **Timeframe**: Higher TF patterns are stronger
4. **Context**: Align with trend direction for continuation, against for reversal
